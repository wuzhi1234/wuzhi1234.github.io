Mahout的安装与配置

编程
  • Hadoop
  • Mahout

Mahout是Hadoop的一种高级应用。运行Mahout需要提前安装好Hadoop。Hadoop的安装网上很多，并不复杂，这里不再讲述，接下来阐述怎么安装Mahout。

下载二进制解压安装。

直接使用homebrew安装mahout
brew install mahout
会提示你安装hadoop，因为我的hadoop不是最新版本所以又下载了2.7.1
下载在/usr/local/Cellar/mahout目录下

启动hadoop，也可以用伪分布式来测试

mahout –help

检查Mahout是否安装完好，看是否列出了一些算法

mahout使用准备

  1. 下载一个文件synthetic_control.data

  2. 启动Hadoop：$HADOOP_HOME/bin/start-all.sh

  3. 创建测试目录testdata，并把数据导入到这个tastdata目录中(这里的目录的名字只能是testdata)

  4. 使用kmeans算法(这会运行几分钟左右)

  5. 查看结果
    hadoop fs -lsr output
    如果看到以下结果那么算法运行成功，你的安装也就成功了。
    clusteredPoints clusters-0 clusters-1 clusters-10 clusters-2 clusters-3 clusters-4 clusters-5 clusters-6 clusters-7 clusters-8 clusters-9 data
